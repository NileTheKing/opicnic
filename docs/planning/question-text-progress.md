DataInitializer의 영어 플레이스홀더 → 실제 OPIc 스타일 영어 문제 교체 추적.

**서비스 컨셉**: OPIc 고득점 목적. 추천 루트(1세트 주제)만 제공.
**참고 자료**: 오픽보물창고 (구조/패턴 파악용, 직접 인용 ✗)
**작업 기준**: 유형별 실제 OPIc 스타일로 직접 작성
- 1번: 묘사 / 2번: 루틴 / 3번: 경험 / 4번: 인상적 경험
- 5번: 인터뷰어에게 질문하기 / 6번: 정보요청 롤플레이 / 7번: 문제해결 롤플레이 / 8번: 유사경험
- 9번: 비교 / 10번: 사회이슈

- `[R]` 참고자료 있음 (책에서 구조 확인됨)
- `[D]` 초안 작성 완료 (DataInitializer 반영)
- `[✓]` 검토 완료
